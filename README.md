# Лендинг для тату-мастера

Современный адаптивный лендинг для тату-мастера с темной темой и фиолетовыми акцентами.

## Технологии

- Чистый HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- SEO-оптимизация

## Структура проекта

```
tatu/
├── index.html          # Главная страница
├── css/
│   └── styles.css     # Все стили
├── js/
│   └── main.js        # Интерактивность (фильтры, слайдер, анимации)
├── favicon.ico        # Иконка сайта
├── package.json       # Зависимости (только gh-pages)
└── deploy.sh          # Скрипт деплоя
```

## Установка и запуск

Проект не требует установки зависимостей для разработки. Просто откройте `index.html` в браузере.

Для деплоя на GitHub Pages:

```bash
# Установка gh-pages (если еще не установлен)
npm install

# Деплой на GitHub Pages
npm run deploy
```

## Деплой на GitHub Pages

### Быстрый деплой

Для деплоя проекта на GitHub Pages используйте команду:

```bash
npm run deploy
```

Эта команда:
1. Создаст временную папку `.deploy`
2. Скопирует только нужные файлы (index.html, css/, js/, favicon.ico)
3. Задеплоит на GitHub Pages через `gh-pages`

### Настройка GitHub Pages

После первого деплоя нужно включить GitHub Pages в настройках репозитория:

1. Перейдите в Settings вашего репозитория на GitHub
2. Найдите раздел "Pages" в боковом меню
3. В "Source" выберите ветку `gh-pages` и папку `/ (root)`
4. Сохраните изменения

Ваш сайт будет доступен по адресу:
**https://ob1-wan-kenobi.github.io/tatu/**

## Компоненты

- **Hero** - Главный экран с кнопкой записи
- **Portfolio** - Галерея работ с фильтрами (JavaScript)
- **About** - О мастере
- **Services** - Услуги и цены
- **Reviews** - Отзывы клиентов (слайдер на JavaScript)
- **Contacts** - Контакты и карта
- **Footer** - Футер с соцсетями

## Функциональность

- **Фильтры портфолио** - фильтрация работ по стилям (Все, Реализм, Минимализм, Олдскул)
- **Слайдер отзывов** - навигация стрелками и точками
- **Анимации при скролле** - плавное появление секций
- **Адаптивный дизайн** - для desktop, tablet, mobile
- **Hover-эффекты** - фиолетовые акценты при наведении

## SEO

Проект полностью оптимизирован для поисковых систем:
- Мета-теги (title, description, keywords)
- Open Graph и Twitter Card
- Структурированные данные JSON-LD (LocalBusiness, Person)
- Правильная структура заголовков (h1, h2)
- Атрибуты alt для всех изображений
- Canonical URL

## Настройка перед деплоем

Перед деплоем замените placeholder данные:

1. **Имя мастера**: замените `[Имя мастера]` во всех файлах
2. **Ссылки**: обновите ссылки на Telegram и Instagram в `index.html` и `js/main.js`
3. **Изображения**: замените placeholder изображения на реальные
4. **Адрес**: обновите адрес студии в секции Contacts
5. **Карта**: обновите iframe с картой Google Maps
6. **SEO**: обновите URL в мета-тегах (если нужен другой домен)

## Адаптивность

- **Desktop**: полная версия со всеми эффектами
- **Tablet** (768px-1024px): адаптированная сетка, уменьшенные отступы
- **Mobile** (<768px): одноколоночная верстка, упрощенные анимации
